# Portfolio Website

**Période :** 2025  
**Statut :** Itératif  
**Type :** Site portfolio — Plusieurs versions

## Stack technique

`Nuxt 3` `Vue 3` `TypeScript` `CSS` `Tailwind` `MeiliSearch` `Docker`

## Problématique

Un portfolio développeur doit évoluer avec les compétences. J'ai exploré plusieurs approches pour mon propre site : de Nuxt 3 simple, à l'ajout d'un moteur de recherche full-text avec MeiliSearch, jusqu'à un frontend dédié.

## Solutions

Trois versions itératives de mon portfolio :

1. **Portfolio Nuxt** (06/2025) — Première version avec Nuxt 3, layout personnalisé
2. **Portfolio MeiliSearch** (07/2025) — Intégration de MeiliSearch pour une recherche full-text des projets et compétences, dockerisé
3. **Portfolio Frontend** (07/2025) — Version raffinée avec architecture Vue 3 propre

## Résultats

- ✅ 3 versions itératives montrant l'évolution technique
- ✅ Recherche full-text ultra-rapide avec MeiliSearch
- ✅ Déploiement avec Docker
- ✅ UI/UX modernes et responsives
