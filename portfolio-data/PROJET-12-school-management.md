# School Management System

**Période :** 2021 – 2022  
**Statut :** Terminé  
**Type :** Application fullstack — Gestion scolaire

## Stack technique

`Laravel` `Vue.js` `Vue 3` `PHP` `JavaScript` `MySQL` `Blade`

## Problématique

Un établissement scolaire a besoin d'une application centralisée pour gérer étudiants, enseignants, cours, notes et plannings. L'approche retenue : une API Laravel + des frontends séparés.

## Solutions

Deux dépôts interconnectés + un projet étudiant :

- **School Management API** (2021) — Backend Laravel complet : gestion des étudiants, enseignants, classes, matières, notes, emplois du temps, API REST
- **Vue.js School Management** (2021) — Frontend Vue.js 2 consommant l'API : dashboard, CRUD étudiants/enseignants, gestion des notes
- **Lasalle** (2022) — Projet scolaire Laravel complet avec authentification, rôles et interface AdminLTE

## Résultats

- ✅ API REST complète avec authentification et autorisations
- ✅ Frontend Vue.js réactif avec gestion d'état
- ✅ Architecture découplée (API + SPA)
- ✅ Projet d'apprentissage fullstack abouti
