# Nuxt Modern Starter

**Période :** 2026
**Statut :** Actif
**Type :** Template — Starter kit full-stack

## Stack technique

`Nuxt 4` `Vue 3` `TypeScript` `Pinia` `OIDC` `Keycloak` `Sentry` `shadcn-vue` `Reka UI` `i18n` `PWA` `Tailwind CSS v4` `OpenAPI`

## Problématique

Lancer un nouveau projet Nuxt full-stack moderne nécessite de configurer manuellement l'authentification OIDC, la gestion des permissions, l'internationalisation, le monitoring, les cookies RGPD, et la PWA — des semaines de configuration répétitive à chaque projet.

## Solution

**Nuxt Modern Starter** est un template GitHub public prêt à l'emploi :

- **Auth OIDC** — Intégration Keycloak avec gestion des rôles et permissions
- **Monitoring** — Sentry intégré pour le suivi des erreurs
- **i18n** — Internationalisation complète avec fichiers de traduction
- **RGPD** — Gestion des cookies avec bannière de consentement
- **PWA** — Support progressive web app
- **UI** — shadcn-vue + Reka UI + Tailwind CSS v4
- **Dashboard** — Layout admin prêt à l'emploi
- **OpenAPI** — Spécification et documentation d'API
