# Auth Solutions & Laravel Boilerplates

**Période :** 2022 – 2025  
**Statut :** Terminé  
**Type :** Starters & templates d'authentification

## Stack technique

`Laravel` `Vue 3` `Livewire` `Sanctum` `AdminLTE` `Vuexy` `PHP` `MySQL`

## Problématique

L'authentification est un socle commun à quasiment toutes les applications web. J'ai exploré différentes approches (JWT, Sanctum, Livewire, templates HTML) pour construire des bases réutilisables.

## Solutions

- **Vue Auth Sanctum** (2025) — Authentification Vue 3 + Laravel Sanctum (SPA, token-based)
- **Vuexy Auth** (2022) — Intégration du template Vuexy dans Laravel 9 pour un UI auth premium
- **Livewire Auth** (2022) — Authentification Laravel 9 + Livewire + AdminLTE (server-side)
- **Laravel 9 Starter Livewire** (2022) — Starter complet avec Livewire, AdminLTE, Docker, roles & permissions
- **Laravel 8 Mini Projet** (2022) — Application de base Laravel 8
- **Laravel Vue.js 3** (2021) — Starter Laravel + Vue 3

## Résultats

- ✅ Plusieurs approches d'auth testées : Sanctum (SPA), Livewire (server), Vuexy (template)
- ✅ Starters prêts à l'emploi pour démarrer des projets plus rapidement
- ✅ Dockerisation pour un environnement de développement reproductible
- ✅ Base de code réutilisable pour les projets futurs
