# Music Player App

**Période :** 2021 – 2022  
**Statut :** Terminé  
**Type :** Application fullstack — Lecteur de musique

## Stack technique

`Vue 3` `Laravel` `PHP` `REST API` `MySQL` `JavaScript` `HTML5` `CSS3`

## Problématique

Créer un écosystème complet de lecture musicale : un backend qui reçoit et indexe des morceaux, plusieurs frontends pour les écouter, et une API REST dédiée.

## Solutions

Quatre projets complémentaires :

- **Vue 3 Music Player** (2022) — Lecteur moderne en Vue 3 avec interface fluide et gestion de playlists
- **HTML Music Player** (2022) — Version allégée en Vanilla JavaScript/VueJS
- **Laravel 9 Music API** (2022) — API REST qui reçoit les morceaux et retourne les métadonnées (titre, artiste, durée, genre)
- **Music Player** (2021) — Version initiale avec Laravel + Blade

## Résultats

- ✅ API complète d'indexation et de métadonnées musicales
- ✅ 3 interfaces différentes (Vue 3, Vanilla JS, Laravel Blade)
- ✅ Architecture REST avec endpoints dédiés
- ✅ Projet d'apprentissage : Vue 3 + Laravel
