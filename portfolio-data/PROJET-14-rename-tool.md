# Rename — Convertisseur d'extensions de fichiers

**Période :** 2021  
**Statut :** Terminé  
**Type :** Outil utilitaire web

## Stack technique

`JavaScript` `PHP` `CSS` `jQuery` `SweetAlert2`

## Problématique

Un besoin simple mais récurrent : pouvoir **changer l'extension** d'un lot de fichiers rapidement (par exemple, convertir des fichiers `.png` en `.jpg`, ou uniformiser des extensions mal nommées).

## Solution

Application web utilitaire qui permet de :

- Sélectionner des fichiers
- Définir une extension source et une extension cible
- Renommer en masse avec confirmation
- Interface simple avec SweetAlert2 pour les confirmations

## Résultats

- ✅ Outil fonctionnel pour le renommage batch de fichiers
- ✅ Interface utilisateur propre et réactive
- ✅ Projet d'apprentissage : PHP + jQuery + manipulation fichiers
