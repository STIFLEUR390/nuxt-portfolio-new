# Fullstack Dev Skills

**Période :** 2026  
**Statut :** Actif  
**Type :** Plugins Claude Code — Kit de développement fullstack

## Stack technique

`Claude Code Plugin System` `Shell` `Laravel` `Nuxt` `NativePHP` `Tauri` `Docker`

## Problématique

Un développeur fullstack jongle avec de nombreux domaines (frontend, backend, mobile, desktop, DevOps, base de données, design). Configurer Claude Code pour chaque domaine et chaque projet prend du temps et casse le flow de travail.

## Solution

Une collection de **7 plugins catégoriels** pour Claude Code, chacun regroupant plusieurs skills spécialisés :

| Plugin | Rôle | Compétences incluses |
|--------|------|---------------------|
| **Frontend** | UI moderne | Nuxt, Vue 3, shadcn-vue, Tailwind |
| **Backend** | Serveur | Laravel 12+, PHP 8.3, API REST/GraphQL, Redis |
| **Cross-Platform** | Multi-plateforme | NativePHP (desktop), Tauri (Rust mobile) |
| **Database** | Données | MySQL, PostgreSQL, optimisation |
| **Design** | Interface | Design systems, tokens, UI/UX |
| **DevOps** | Infra | Docker, CI/CD, orchestration |
| **Practices** | Qualité | TDD/BDD, OWASP, Git, code review, debug |

## Résultats

- ✅ 7 plugins couvrant l'ensemble du développement fullstack
- ✅ Installation en une commande depuis la Marketplace Claude Code
- ✅ Skills spécialisés par domaine pour des prompts plus précis
- ✅ Documentation complète avec exemples
