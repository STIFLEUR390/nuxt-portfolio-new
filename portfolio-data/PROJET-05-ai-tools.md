# AI Tools Platform

**Période :** 2025 – 2026  
**Statut :** Actif  
**Type :** Plateforme de documentation & outils AI

## Stack technique

`Nuxt` `TypeScript` `MDX` `Mintlify` `Vue` `AI APIs`

## Problématique

L'écosystème des outils AI (Claude Code, Cursor, Windsurf) évolue rapidement. Les développeurs ont besoin d'une documentation centralisée, d'exemples concrets et d'outils pour maximiser leur productivité avec ces assistants.

## Solution

Un ensemble de ressources pour l'écosystème AI :

- **Nuxt AI Tool** (2025) — Application Nuxt avec interface de type Cursor Rules, permettant d'explorer et gérer des règles et configurations pour les AI coding assistants
- **AI Tools Documentation** (2025) — Documentation complète sur Claude Code, Cursor et Windsurf avec Mintlify, couvrant les bonnes pratiques, workflows et astuces avancées
- **Documentation technique** — Architecture API, guides essentiels, snippets de code réutilisables

## Résultats

- ✅ Documentation structurée et navigable pour 3 outils AI majeurs
- ✅ Interface pratique pour la gestion de règles AI (Cursor Rules)
- ✅ Ressources adaptées aux développeurs fullstack utilisant l'IA
