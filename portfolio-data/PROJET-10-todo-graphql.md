# Todo GraphQL & Task Management

**Période :** 2023  
**Statut :** Terminé  
**Type :** Application fullstack — Gestion de tâches

## Stack technique

`Laravel` `GraphQL` `Vue 3` `PHP` `MySQL` `Blade` `Vite`

## Problématique

Explorer **GraphQL** comme alternative à REST pour la gestion de données. L'objectif était de construire une application de gestion de tâches complète avec un endpoint GraphQL unique, flexible et fortement typé.

## Solution

Trois dépôts complémentaires :

- **Todolist-Graphql** (2023) — Backend Laravel avec API GraphQL complète (queries, mutations, subscriptions)
- **todolit-api** (2023) — API REST Laravel pour la gestion des tâches
- **todolit-vue** (2023) — Frontend Vue 3 avec Composition API

## Résultats

- ✅ API GraphQL flexible avec requêtes personnalisées
- ✅ Frontend Vue 3 avec gestion d'état réactive
- ✅ Architecture fullstack complète (API + Frontend)
- ✅ Deux approches : REST et GraphQL comparées
